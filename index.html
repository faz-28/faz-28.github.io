Summary

I'm an Automation and BMS engineer building the HMI graphics that let technicians see and control a building from one screen. My current work is on Honeywell Experion PKS / EBI, turning client hardware drawings into live HMIWeb displays for airports, hospitals, and commercial buildings — binding every data point so the screen reflects what's actually happening on site.
